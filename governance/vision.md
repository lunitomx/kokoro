# Solution Vision: RaizAncestral (Kokoro)

> El corazón estratégico de Eduardo Muñoz Luna — donde la sabiduría ancestral
> se encuentra con la estrategia de marketing contemporánea.

## Identity

### Description

Kokoro es un sistema de 46+ skills para Claude Code que digitaliza la metodología
de consultoría estratégica de Eduardo Muñoz Luna. Guía emprendedores a través de
un proceso orgánico de 4 fases (Preparar el Suelo, Elegir la Semilla, Germinar,
Cosechar) con herramientas transversales de analytics, ads, creativos y CRM.
Distribuido como repositorio público que se clona y ejecuta directamente en Claude Code.

## Outcomes

| **Outcome** | **Description** |
|-------------|-----------------|
| **Escala de consultoría** | Permitir que la metodología de Eduardo atienda emprendedores sin requerir su presencia directa en cada sesión |
| **Profundidad sobre velocidad** | Cada interacción refleja la voz, filosofía y rigor del Guardián de la Riqueza — nunca tips genéricos |
| **Proceso orgánico completo** | Un emprendedor puede recorrer las 4 fases con herramientas específicas para cada etapa, sin saltar pasos |
| **Datos conectados** | Integración con Meta Ads, GA4, Google Ads, Search Console, Pipedrive para decisiones basadas en evidencia real |
| **Distribución accesible** | Cualquier persona con Claude Code puede clonar el repo y comenzar su primera consulta profunda |
