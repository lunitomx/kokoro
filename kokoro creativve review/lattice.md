## Abstract
Meta Lattice is the architectural unification layer of Meta's advertising system. It replaces the legacy system of fragmented, task-specific models (e.g., one model for Reels, another for Feed) with a single, massive high-capacity model that learns across all surfaces, formats, and objectives simultaneously.

## The "Why": Fragmented Intelligence
In the past, Meta operated like a collection of small libraries:
- **Art Library**: Handled Reels.
- **History Library**: Handled Feed.
- **Math Library**: Handled Click objectives.
The problem was that lessons learned in "Art" didn't benefit "History." Lattice creates one **Giant Library** where every interaction anywhere in the ecosystem smarter.

## Architecture: Multi-Task Learning (MTL)
Lattice uses a transformer-based architecture optimized for high-dimensional data across:
1. **Surfaces**: Facebook Feed, Instagram Stories, Reels, WhatsApp.
2. **Formats**: Static images, Video, Carousel, Text.
3. **Objectives**: Awareness, Clicks, App Installs, Catalog Sales, Leads.

### Technical Mechanism:
- **Shared Parameters**: ~90% of the model is shared across all tasks, allowing "Cross-Objective Knowledge Transfer."
- **Task-Specific Heads**: The final 10% of the model specializes for specific goals (e.g., predicting a "Purchase" vs. a "Like").

## Strategic Playbook: Leveraging the Library
1. **Enable All Placements**: Restricting placements (e.g., "Manual Feed Only") disables Lattice's ability to borrow intelligence from other surfaces. **Broad is mandatory.**
2. **Format Fluidity**: Use the same asset in multiple formats. Lattice identifies which format performs best for a specific user based on global patterns.
3. **Omni-Objective Awareness**: Lattice knows that a user who clicks often might eventually buy. It uses "Low-Intent" signals to find "High-Intent" buyers.

## Ecosystem Integration
- **GEM (The Brain)**: Lattice provides the "Library" of data GEM uses to predict sequence intent.
- **Andromeda (Retrieval)**: Lattice's organized categories help Andromeda navigate the Hierarchical Index faster.
- **Sequence Learning**: Supplies the temporal data that Lattice unifies into a single user profile.
