# /kokoro-luxury-communication — Comunicacion que Eleva

> Modulo Lux by Kokoro: Construccion de Marca
> "El lujo no grita: susurra. No explica: seduce. No vende: enamora."

## Contexto

Guia al invitado en la elevacion de su comunicacion usando codigos
visuales, estrategia de color, artificacion y vocabulario luxurizante.
Lee `lux-communication.md`.

### Resolucion de invitado

Busca en `.kokoro/clients.json`. Verifica `positioning_tier` = luxury/premium.

## Instrucciones para la sesion

### Estrategia del Proyector

> "La forma de comunicar define tu categoria mas que la creacion misma.
> Un mismo objeto puede ser masivo, premium o lujo segun como se cuente
> su historia. Veamos como contar la tuya."

### Proceso — 5 ejercicios

**Ejercicio 1: Paleta 60/30/10**
Definir la paleta cromatica del invitado:
- 60% color dominante — atmosfera emocional principal
- 30% color secundario — contraste y profundidad
- 10% color acento — foco emocional y poder narrativo

Evaluar: ¿La paleta actual comunica los valores del invitado?

**Ejercicio 2: Audit de Codigos Visuales**
Evaluar los 6 principios en la comunicacion actual:

| Principio | Estado actual | Accion |
|-----------|:------------:|--------|
| White Space | | ¿Hay espacio para respirar? |
| Centralidad Visual | | ¿El foco esta claro? |
| Mirada desde Arriba | | ¿Hay distancia estetica? |
| B&N Estrategico | | ¿Usas atemporalidad? |
| Fotografia Cuidada | | ¿Iluminacion, textura, encuadre? |
| Lenguaje Simbolico | | ¿Metaforas o features? |

**Ejercicio 3: Nivel de Artificacion**
Evaluar en que nivel esta y cual es el siguiente:

1. Patrocinio — Apoyar eventos artisticos
2. Colaboracion — Trabajar con artistas
3. Exposiciones propias — En espacios del invitado
4. Premios/Becas — Impacto cultural
5. Fundaciones — Institucionalizar arte-marca
6. Colecciones privadas — Poder simbolico
7. Mecenazgo corporativo — Tejido cultural

**Ejercicio 4: Audit de Vocabulario**
Revisar comunicacion actual del invitado y sustituir:
- Precio → Inversion
- Producto → Creacion
- Gratis → Cortesia
- Descuento → Condiciones especiales
- Cliente → Invitado

**Ejercicio 5: Plan de Comunicacion Espacial**
Definir como los espacios del invitado comunican:
- Eventos simbolicos (no presentaciones de producto)
- Contextos artisticos (no salas de conferencia)
- Rituales de marca (no transacciones)
