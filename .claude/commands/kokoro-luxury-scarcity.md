# /kokoro-luxury-scarcity — Escasez Estrategica

> Modulo Luxelling: Superpoder #1
> "La marca escoge al invitado, no al reves."

## Contexto

Guia al invitado en el diseno de estrategias de escasez que eleven la
deseabilidad sin sacrificar rentabilidad. Lee `luxelling-scarcity.md`.

### Resolucion de invitado

Busca en `.kokoro/clients.json`. Verifica `positioning_tier` = luxury/premium.
Si no tiene tier, sugiere `/kokoro-luxury-assess` primero.

## Instrucciones para la sesion

### Estrategia del Proyector

> "La escasez es el primer superpoder del lujo. No se trata de tener
> poco — se trata de que lo poco que tienes sea extraordinariamente
> deseado. Exploremos como activar este superpoder en tu creacion."

### Proceso — 4 ejercicios

**Ejercicio 1: Dream Equation Audit**
Evaluar la ecuacion actual: Deseabilidad = 1 / (Puertas x Visibilidad)
- Cuantas "puertas" tiene el invitado (puntos de acceso)
- Cual es la visibilidad actual
- Calcular Demand Gap Index: (Demanda - Oferta) / Demanda >= 30%

**Ejercicio 2: Tacticas de Escasez**
Evaluar cuales aplican al invitado:

| Tactica | Descripcion | Aplicable? |
|---------|-------------|:----------:|
| Demand Gap | Mantener demanda 30%+ sobre oferta | |
| Ediciones limitadas | Lotes finitos con storytelling temporal | |
| Distribucion selectiva | <=20% puntos de venta potenciales | |
| Pop-ups efimeros | Activaciones 2-12 semanas | |
| Discontinuidad | Retirar creacion exitosa para elevar valor | |

**Ejercicio 3: Plan de Implementacion**
Para cada tactica seleccionada, definir:
- Que hacer concretamente
- Metrica Lean para medir impacto
- Timeline de implementacion

**Ejercicio 4: Selectivity Ratio**
Definir el ratio de selectividad del invitado:
- Benchmark Gucci: < 600 puntos de acceso
- Benchmark Chanel: < 200 puntos de acceso
- El invitado: ¿cual es su numero ideal?

## Casos de estudio para referencia

- Nike x Dior Air Jordan: 8,500 pares vs 5M pedidos
- Chanel pandemia: 5 incrementos de precio = mas demanda
- Patek Philippe Nautilus: discontinuidad → precio 6x en secundario
- Prada Dubai: rechazo de 3a boutique para proteger Dream Equation
