# Diversificacion Creativa — Referencia para /kokoro-creative-review

> "La diversidad creativa no es un lujo — es el combustible que alimenta
> a los 4 sistemas de IA de Meta."

## Por que la Diversificacion es Obligatoria

En la era de Andromeda y GEM, el targeting manual fue reemplazado por
**Creative Targeting**. La diversificacion creativa es la practica de crear
anuncios con temas, mensajes y visuales DISTINTOS para:
1. Alcanzar diferentes segmentos de audiencia
2. Dar al algoritmo un "Campo de Accion" mas amplio
3. Evitar fatiga creativa (cuando CPAs suben, Andromeda agoto clusters)

## Framework de 3 Pilares

### Pilar 1 — Los 16 Deseos Humanos Basicos (Steven Reiss)

Toda decision de adquisicion esta motivada por uno o mas de estos deseos.
Diversificacion efectiva requiere rotar entre al menos 3-5:

| Deseo | Enfoque del creativo |
|-------|---------------------|
| **Aceptacion** | Aprobacion social, prueba social |
| **Curiosidad** | Misterio, informacion nueva, "Como hacer..." |
| **Alimentacion** | Sabor, textura, placer culinario |
| **Familia** | Legado, proteccion, hijos |
| **Honor** | Valores, etica, tradicion |
| **Idealismo** | Justicia social, proposito |
| **Independencia** | Autonomia, autosuficiencia |
| **Orden** | Organizacion, estructura, paz |
| **Actividad fisica** | Movimiento, energia, salud |
| **Poder** | Estatus, influencia, liderazgo |
| **Romance** | Belleza, conexion, atraccion |
| **Ahorro** | Valor, acumulacion, frugalidad |
| **Contacto social** | Pertenencia, amistad |
| **Estatus social** | Prestigio, reconocimiento |
| **Tranquilidad** | Seguridad, anti-ansiedad |
| **Competencia** | Victoria, rivalidad |

### Pilar 2 — Los 5 Niveles de Consciencia (Eugene Schwartz)

Mapear el mensaje del creativo a la distancia del usuario respecto a la
decision de adquirir:

| Nivel | Estado | Angulo |
|-------|--------|--------|
| 1. Inconsciente | No sabe que tiene un reto | DOLOR / EVITAR |
| 2. Consciente del reto | Sabe el reto, no las soluciones | DOLOR / EVITAR |
| 3. Consciente de soluciones | Sabe que existen, no conoce tu creacion | GANANCIA / LOGRAR |
| 4. Consciente de tu creacion | Conoce tu creacion, necesita confianza | GANANCIA / LOGRAR |
| 5. Listo para decidir | Listo para elegir, necesita impulso | GANANCIA / LOGRAR |

### Pilar 3 — La Matriz de Diversificacion Creativa

Genera una Matriz de 30 Hooks (6 filas x 5 columnas):
- **Filas:** 3 Deseos x 2 tipos de Perfil
- **Columnas:** 5 Niveles de Consciencia

Cada celda = 1 hook unico = 1 cluster diferente en Andromeda.

## Niveles de Sofisticacion de Mercado (Schwartz)

| Nivel | Estrategia | Ejemplo |
|-------|-----------|---------|
| **1 — Primero** | Promesa directa | "Aprende Meta Ads" |
| **2 — Expandir** | Mas grande/rapido | "Duplica tus ventas con Meta Ads" |
| **3 — Mecanismo** | El COMO diferencia | "El sistema de 3 pasos para duplicar ventas" |
| **4 — Mec. Expandido** | El mas actual/especifico | "El NUEVO sistema de IA que triplica ventas" |
| **5 — Identificacion** | Identidad del prospecto | "Para emprendedores cansados de quemar dinero" |

## Despliegue Practico

1. **Seleccion de Hook:** Los primeros 3 segundos definen el "Cluster" en Andromeda
2. **Diversificacion de formato:**
   - **Video (9:16):** Storytelling, senales de intencion profundas
   - **Estaticos:** Hooks de alto impacto, extraccion rapida de features
   - **Secuencias/Carruseles:** Educativo, consideracion mas profunda
3. **Test del Campo de Accion:** Si los CPAs suben = fatiga creativa. Lanzar
   una nueva fila Deseo/Perfil reinicia el ciclo de exploracion

---

## Sinergias con el Ecosistema

- **Andromeda:** Diversificacion provee la "Diversidad de Entidades" necesaria
  para conquistar diferentes clusters del indice
- **GEM:** Mensajes variados proveen "Sequence Delta" para que el modelo
  prediga nueva intencion
- **Advantage+:** Funciona mejor con 10-15+ IDs de creativos diversos
  para testear simultaneamente
