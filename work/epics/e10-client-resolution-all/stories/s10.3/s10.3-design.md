# S10.3 Design — Phase 3+4 Client Resolution

## Approach
Insert the standardized "Resolucion de invitado" section (from kokoro-diagnose.md)
into each of the 8 target skills. The section goes after "Contexto previo" and before
"Instrucciones para la sesion".

## Section Content (canonical from kokoro-diagnose.md)
The section uses "invitado" vocabulary consistently and references:
- `.kokoro/clients.json` for graph lookup
- `find_by_name` for partial, case-insensitive matching
- Graceful fallback when no client graph exists
- Suggestion to register via `/kokoro-client` when appropriate

## Integration Points
Each skill already has a "Contexto previo" subsection under "## Contexto".
The new section is added as a peer subsection `### Resolucion de invitado`.
