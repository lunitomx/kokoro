# S10.4 — MEMORY.md Auto-Update

**Epic:** E10 (client-resolution-all)
**Size:** S
**Status:** In Progress

## User Story

When Kokoro client sync generates or updates reference files, the MEMORY.md
index should be updated automatically with new entries. Existing entries
are preserved (no duplicates).

## Acceptance Criteria

1. `update_memory_index(memory_md, new_files)` adds entries under `## References`
2. Skips files already listed in MEMORY.md
3. Returns `True` if entries were added, `False` otherwise
4. Handles missing MEMORY.md and empty file lists gracefully
5. Inserts before `## Feedback` section when present
