# S10.4 Design

## Approach

Add `update_memory_index(memory_md: Path, new_files: list[Path]) -> bool` to
`src/kokoro/clients/sync.py`.

### Logic
1. Early return `False` if MEMORY.md missing or file list empty
2. For each reference file, check if filename already in MEMORY.md content
3. If not present, generate entry line: `- [Title](filename) — auto-generated from client graph`
4. Insert before `## Feedback` if present, else append to `## References`
5. Write updated content, return `True` if any added

### Title extraction
`reference_konecta_park.md` -> strip prefix/suffix -> replace `_` with space -> title case -> "Konecta Park"
