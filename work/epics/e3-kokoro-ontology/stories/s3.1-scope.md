# S3.1: Ontology Schema — Scope

> **Status:** In Progress

## In Scope

- `src/kokoro/ontology/__init__.py` — package init
- `src/kokoro/ontology/models.py` — all Pydantic models

- EdgeType enum: 5 types (alimenta, valida, experimenta, mide, pertenece_a)

- Unit tests for all models (serialization, validation, round-trip)
