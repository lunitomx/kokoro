# S11.2 Plan — /kokoro-connect

## Tasks

### Task 1: Create skill file + knowledge file

**Action:**
1. Write RED tests in `tests/test_output_structure.py`:

   - Add `"kokoro-connect"` to `VALID_COMMANDS` set
   - Add `"kokoro-connect"` to `EXISTING_COMMANDS` set
   - Run tests — confirm failures (files don't exist yet)
2. Create `.claude/commands/kokoro-connect.md` — main skill file:
   - 5-step flow: Client Resolution → Platform Discovery → Account Selection → Persist Mapping → Confirmation

   - Stores in `ClientProfile.metadata["platform_accounts"]` with keys: `meta_ads`, `google_ads`, `ga4`, `gsc`

   - Graceful degradation for unreachable MCP servers
   - Output template with connection summary table
3. Create `.claude/knowledge/kokoro-connect-platforms.md` — platform reference:
   - Table of platforms, MCP servers, discovery tools, account ID formats
   - Key conventions for each platform
4. Run tests — confirm GREEN

**Verification:**
- `pytest tests/test_output_structure.py` passes
- Skill file contains output template with required structural elements
- Knowledge file documents all 4 platforms
