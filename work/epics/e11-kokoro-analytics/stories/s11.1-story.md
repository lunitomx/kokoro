# S11.1 — MCP Registration

## User Story (Connextra)

**As** Eduardo (Kokoro operator),
**I want** mcpfbads and mcpgoogleads MCP servers registered in Claude Code,
**So that** all 4 MétricaRadix MCP servers (FB Ads, Google Ads, GA4, GSC) are
accessible from Kokoro for analytics queries.

## Acceptance Criteria

- mcpfbads server is registered in Claude Code MCP configuration
- mcpgoogleads server is registered in Claude Code MCP configuration
- All 4 MCP servers respond when called from Claude Code:
  - google-analytics (already registered)
  - google-search-console (already registered)
  - mcpfbads (new)
  - mcpgoogleads (new)
- Tools from mcpfbads appear in Claude Code's available tool list
- Tools from mcpgoogleads appear in Claude Code's available tool list
