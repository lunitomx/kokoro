# S11.1 — Scope

## In Scope

- Register mcpfbads MCP server in Claude Code configuration
- Register mcpgoogleads MCP server in Claude Code configuration
- Verify all 4 servers respond (FB Ads, Google Ads, GA4, GSC)

## Out of Scope

- Modifying MétricaRadix code
- Creating new Kokoro skills
- Client graph integration
- Credential management or rotation
- Multi-account configuration

## Done Criteria

- mcpfbads tools appear in Claude Code's tool list
- mcpgoogleads tools appear in Claude Code's tool list
- All 4 MCP servers can be called from Claude Code
