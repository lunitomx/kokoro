### Task 2: GREEN — Create skill file + knowledge file

**Action:**
1. Create `.claude/commands/kokoro-analytics.md` — main skill file:
   - Query intent extraction: client name, platforms, date range, depth
   - Client resolution: search `.kokoro/clients.json` by name, fallback to active session
   - Platform account lookup: `ClientProfile.metadata["platform_accounts"]`
   - MCP tool dispatch per platform (meta_ads, google_ads, ga4, gsc)
   - Cross-platform overview when user asks for general performance
   - Natural language date range interpretation (default: last 7 days)
   - Graceful handling: missing platforms -> suggest `/kokoro-connect`, unreachable servers -> report and continue
   - Kokoro voice: luxurizante vocabulary, Proyector strategy, metric translation
   - Output template with per-platform sections and summary
2. Create `.claude/knowledge/kokoro-analytics-metrics.md` — reference file:
   - Platform-to-MCP tool mapping table (overview + detail tools per platform)
   - MCP tool parameter reference (account_id, date_range, property_id formats)
   - Metric translation glossary (CTR, CPC, CPM, ROAS, impressions, clicks, bounce rate)
   - Date range interpretation guide (este mes, semana pasada, ayer, etc.)
3. Run `pytest tests/test_output_structure.py` — confirm GREEN

**Files:**
- `.claude/commands/kokoro-analytics.md` (create)
