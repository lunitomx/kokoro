# Scope: S11.4 — Traduccion de metricas

## In Scope

- Expand `kokoro-analytics-metrics.md` glossary from 12 to comprehensive coverage
- Meta Ads metrics: spend, reach, frequency, cpc, cpm, ctr, impressions, clicks,
  actions, cost_per_action, purchase_roas, video views, engagement metrics
- Google Ads metrics: cost, conversions, conv_rate, cost_per_conversion,
  search_impression_share, quality_score, avg_cpc, ctr, impressions, clicks
- GA4 metrics: sessions, users, newUsers, bounceRate, engagementRate,
  averageSessionDuration, screenPageViews, conversions, eventCount,
  ecommerce metrics (revenue, transactions, itemRevenue)
- Search Console metrics: clicks, impressions, ctr, position
- Each entry: technical name, business meaning, Kokoro translation template,
  highlight thresholds, related metrics
- Unknown metric fallback protocol
- Extension mirror sync

## Out of Scope

- Modifying the skill file (`kokoro-analytics.md`)
- Modifying MCP server code
- Adding new MCP tools or parameters
- Python code changes
- Automated threshold detection

- [ ] Glossary covers all metrics from 4 MCP servers
- [ ] Each entry has 5 fields (name, meaning, template, threshold, related)
- [ ] Unknown metric protocol documented
