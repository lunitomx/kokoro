# Scope: S11.5 — /kokoro-scorecard

## In Scope

- `/kokoro-scorecard` Claude Code skill (prompt file)
- Client resolution from name (same pattern as /kokoro-analytics)
- Query ALL connected platforms via MCP tools (parallel dispatch)
- Unified scorecard output with:
  - Header: invitado name, period, platforms connected
  - Per-platform summary: key metrics with trend vs previous period
  - Cross-platform insights: traffic sources, cost efficiency
  - Health indicators: green/yellow/red per platform
  - Recommended actions based on data patterns
- Graceful degradation when MCP servers are unreachable
- Extension mirror at extension/.claude/commands/kokoro-scorecard.md
- Test coverage for skill structure and output template
- Registration in COACHING_SKILLS and VALID_COMMANDS lists

## Out of Scope

- Modifying kokoro-analytics-metrics.md (S11.4 parallel work)
- Campaign modification or optimization actions (read-only)
- Credential management or token refresh
- Web UI or dashboard — purely conversational
- Historical data storage or caching
- Modifying MCP server code (MetricaRadix)
