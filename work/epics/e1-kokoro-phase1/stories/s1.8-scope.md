## In Scope
- `.claude/commands/kokoro.md` router skill file (markdown prompt)
- `.claude/commands/kokoro-session.md` session management skill file (markdown prompt)
- Router logic: diagnostic questions to identify user's Phase 1 position
- Router awareness of all 4 Phase 1 skills and methodology order

- Session continue: review completed work, identify next step
- Session review: structured Phase 1 progress summary
- CLI integration: `kokoro init` copies both new skill files

- Persistent session state across Claude Code sessions (native sessions suffice)
- Automatic detection of completed skills (user self-reports progress)
- Changes to existing Phase 1 skills (S1.4-S1.7 are done)
- Web UI or dashboard for progress tracking
