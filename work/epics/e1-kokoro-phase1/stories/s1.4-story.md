# S1.4 — /kokoro-diagnose

## User Story

**As** an entrepreneur using Kokoro,
**I want** a guided diagnostic session (Speed Boat + Vision 20/20),
**So that** I can identify my business's current obstacles, advantages, risks, and vision.

## Acceptance Criteria

### AC1: Skill File Exists
```gherkin
Given the kokoro extension is installed
When I check .claude/commands/
Then kokoro-diagnose.md exists as a valid skill file
```

### AC2: Speed Boat Exercise
```gherkin
Given I run /kokoro-diagnose
When the session begins
Then Claude guides me through the Speed Boat exercise
And asks about anchors (obstacles), wind (advantages), and rocks (risks)
```

### AC3: Vision 20/20 Exercise
```gherkin
Given the Speed Boat exercise is complete
When the session continues
Then Claude guides me through Vision 20/20
And asks about retrospective vision and prospective vision
```

### AC4: Diagnostic Summary
```gherkin
Given both exercises are complete
When the session ends
Then Claude produces a structured diagnostic summary
And the summary includes findings from both exercises
```

### AC5: CLI Integration
```gherkin
Given the kokoro package is installed
When I run kokoro init
Then the diagnose skill file is copied to .claude/commands/
```

### AC6: Tests
```gherkin
Given the skill implementation
When tests run
Then all skill-related tests pass
And the skill file content is validated
```
