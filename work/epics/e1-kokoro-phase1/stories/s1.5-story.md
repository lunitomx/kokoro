# S1.5 — /kokoro-mountain

## User Story

**As** an entrepreneur using Kokoro,
**I want** a guided session to build my Montaña del Mañana and define OKRs,
**So that** I can translate my 3-year vision into measurable quarterly objectives.

## Acceptance Criteria

### AC1: Skill File Exists
Given the kokoro extension is installed
When I check .claude/commands/
Then kokoro-mountain.md exists as a valid skill file

### AC2: Montaña del Mañana Exercise
Given I run /kokoro-mountain
When the session begins
Then Claude guides me through building the Montaña del Mañana
And asks about the summit (3-year vision), base camps (1-year, 6-month hitos), and the base (current state)

### AC3: OKR Definition
Given the Montaña is built
When the session continues
Then Claude guides me through defining 2-3 OKRs with measurable Key Results

### AC4: Vision Summary
Given both exercises are complete
When the session ends
Then Claude produces a structured vision summary with mountain map and OKRs

### AC5: CLI Integration
Given the kokoro package is installed
When I run kokoro init
Then the mountain skill file is copied to .claude/commands/

### AC6: Tests
Given the skill implementation
When tests run
Then all skill-related tests pass
