# S1.7 — /kokoro-finance

## User Story

**As** an entrepreneur using Kokoro,
**I want** a guided financial assessment session,
**So that** I can understand the real costs, conversion rates, profitability, and maximum acquisition budget for each business line, making marketing an investment rather than a gamble.

## Acceptance Criteria

### AC1: Skill File Exists
Given the kokoro extension is installed
When I check .claude/commands/
Then kokoro-finance.md exists as a valid skill file

### AC2: Financial Inventory
Given I run /kokoro-finance
When the session begins
Then Claude guides me through a financial inventory per product/service
And covers revenue, direct costs, indirect costs, real margin, and active clients

### AC3: Acquisition Analysis
Given the financial inventory is complete
When the session continues
Then Claude guides me through acquisition analysis (CPA, channels, conversion rates)

### AC4: Key Metrics
Given the session is in progress
When financial data is discussed
Then Claude references key metrics (CPA, ROI, conversion rate, LTV)
And explains the "client = 2+ purchases" definition

### AC5: Budget and Validation Plan
Given the analysis is complete
When the session concludes
Then Claude produces a 90-day investment plan with metrics to monitor
And references sector-specific budget benchmarks

### AC6: CLI Integration
Given the kokoro package is installed
When I run kokoro init
Then the finance skill file is copied to .claude/commands/

### AC7: Tests
Given the skill implementation
When tests run
Then all skill-related tests pass
