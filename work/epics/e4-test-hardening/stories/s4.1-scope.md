# S4.1 Scope — Conftest Consolidation

> **Story:** S4.1 | **Epic:** E4 | **Size:** S

## User Story

As a developer maintaining Kokoro tests,
I want all test files to use shared conftest fixtures
So that a path change propagates everywhere instead of editing 10 files.

## In Scope

- Migrate 10 test files from EXTENSION_DIR module constant to conftest fixtures
- Add `skill_path` fixture factory or per-skill fixtures to conftest.py
- Remove all EXTENSION_DIR / SKILL_PATH module-level constants from test files
- Zero regressions in 263 tests

## Out of Scope

- New test assertions (that's S4.2/S4.3)
- Changing test class structure
- Adding new test files

## Done When

- Zero test files define EXTENSION_DIR
- All tests use conftest fixtures for paths
- 263/263 tests pass
- pyright + ruff clean
