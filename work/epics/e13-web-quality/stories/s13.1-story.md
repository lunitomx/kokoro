# Story: Skill /kokoro-audit

## User Story
As Eduardo's client (a non-technical entrepreneur),
I want to audit my website's quality through Kokoro,
so that I get actionable recommendations in language I understand.

## Acceptance Criteria

### Scenario: Run audit on client site
```gherkin
Given a client with a website URL
When I invoke /kokoro-audit
Then it runs performance, SEO, accessibility checks
And presents results in non-technical language with Kokoro voice
And credits Addy Osmani's web-quality-skills
```

## Notes
- Wraps existing skills: seo-technical, seo-content, seo-page, performance, accessibility, best-practices
- Credit: "Powered by web-quality-skills by Addy Osmani (https://github.com/nicholasgriffintn/web-quality-skills)"
- Translates technical findings to business impact
- Integrates with client graph (audit client's registered site)
