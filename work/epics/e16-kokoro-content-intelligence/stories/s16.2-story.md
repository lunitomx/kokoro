# S16.2 — /kokoro-intel: YouTube Competitive Intelligence

## Objective

Create a skill that searches YouTube by keyword, downloads and transcribes
the top N results using the kokoro-listen pipeline, then analyzes ALL
transcriptions together to produce a competitive intelligence report with
content opportunities ranked by impact.

## Value

- Eduardo stops guessing what content to create -- data shows what's missing
- Analyzing 5-10 videos costs ~$8-16 MXN total
- Content strategy grounded in evidence, not intuition
- Reusable for any niche or invitado

## Acceptance Criteria

- [ ] Skill accepts YouTube search query and analyzes top N results
- [ ] Skill accepts specific URLs for targeted analysis
- [ ] Uses kokoro-listen pipeline for download/transcribe
- [ ] Analyzes patterns: topics, structure, tone, duration, gaps, angles
- [ ] Presents competitive intelligence report
- [ ] Proposes content opportunities ranked by impact
- [ ] Estimates cost before running
- [ ] Follows Kokoro vocabulary and Proyector strategy
