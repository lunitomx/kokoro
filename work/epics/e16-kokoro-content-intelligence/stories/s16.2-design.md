# Design: S16.2 — /kokoro-intel

## Overview

/kokoro-intel is a competitive intelligence skill that searches YouTube for
a keyword, downloads and transcribes the top results using the kokoro-listen
pipeline, then analyzes all transcriptions together to find patterns, gaps,
and content opportunities.

## Skill Structure

Following the pattern established by kokoro-listen, kokoro-ads, and
kokoro-research:

```
.claude/commands/kokoro-intel.md
```

The skill is a prompt file that instructs Claude what to do. No Python code.

## Two Modes of Operation

### Mode 1: Search Mode (primary)

User provides a search query. The skill:
1. Estimates cost based on N videos
2. Searches YouTube via yt-dlp: `ytsearch{N}:{query}`
3. Downloads audio of top N results
4. Transcribes each with Whisper API (kokoro-listen pipeline)
5. Analyzes ALL transcriptions together
6. Presents competitive intelligence report

### Mode 2: URL Mode

User provides specific YouTube URLs. The skill:
1. Estimates cost based on number of URLs
2. Downloads and transcribes each URL
3. Analyzes all together
4. Presents report

## Analysis Framework

The analysis examines 6 dimensions across all transcriptions:

1. **Common Topics** — What everyone covers (table stakes)
2. **Structure Patterns** — How they organize content (intro, sections, CTA)
3. **Tone and Style** — Professional, casual, tutorial, storytelling, etc.
4. **Duration Patterns** — Short vs long, what topics get more time
5. **GAPS** — What nobody covers or covers poorly
6. **Angles** — What perspectives are missing (beginner, advanced, case study)

## Output Format

### Per-Video Summary Table

| # | Title | Duration | Topics | Style |
|---|-------|----------|--------|-------|

### Pattern Analysis (6 dimensions)

Narrative analysis for each dimension with specific evidence from
transcriptions.

### Opportunities Report

Ranked list of content opportunities with:
- Topic + angle
- Why it's an opportunity (gap/weakness/missing perspective)
- Estimated impact (high/medium/low)
- Suggested format (horizontal, short, or both)

## Cost Estimation

Before executing, show:
- Number of videos to analyze
- Estimated total duration
- Estimated Whisper cost (duration x $0.006 USD/min)
- Estimated MXN cost (USD x 18)

Ask for confirmation if cost > $1 USD (~$18 MXN).

## Integration Points

- **kokoro-listen**: Reuse the exact download + transcribe pipeline
- **kokoro-research**: Complementary — research does web, intel does video
- **S16.3 (future)**: Will extend the gap analysis
- **S16.4 (future)**: Will use opportunities to generate calendar

## Design Decisions

### DD-1: Summarize before cross-analysis

Long transcriptions can saturate Claude's context. Each transcription
gets summarized to ~500 words before cross-analysis. Full transcriptions
are saved to disk for reference.

### DD-2: Sequential processing, not parallel

Process videos one at a time to respect YouTube and Whisper rate limits.
Show progress as each video completes.

### DD-3: Default 5 videos, max 10

5 videos gives good signal at ~$8 MXN. User can request up to 10 for
deeper analysis at ~$16 MXN.
