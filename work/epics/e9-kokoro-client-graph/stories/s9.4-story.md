# Story: Integración de client graph con skills existentes

## User Story
As Kokoro,
I want existing skills to automatically read the client registry and load context,
so that when Eduardo works on a client project, the skill already knows who the client is.

## Acceptance Criteria

### Scenario: kokoro-ads reads client context
```gherkin
Given a client "Konecta Park" exists in the registry with context_file set
When /kokoro-ads is invoked
Then the skill reads the client profile and context file before generating copy
```

### Scenario: Client not found
```gherkin
Given no matching client in the registry
When a skill tries to resolve client
Then it asks Eduardo for client details (graceful fallback)
```
