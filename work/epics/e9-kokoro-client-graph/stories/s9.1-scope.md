# S9.1 Scope — Modelo de datos para Client Graph

## In Scope

- `src/kokoro/clients/__init__.py` — module init with public exports
- `src/kokoro/clients/models.py` — ClientProfile and ClientRegistry Pydantic models
- All fields from epic architecture: id, name, group, description, repos, campaign_folder, context_file, segments, industry, coaching_state_path, created, updated, metadata
- ClientRegistry with `find_by_name()` method (case-insensitive, substring match)
- Type annotations compatible with pyright strict
- Unit tests for model creation, validation, and registry search

- CLI skill (S9.3)
- Integration with existing skills (S9.4)
- Memory sync (S9.5)
- Store/load from disk — models only
