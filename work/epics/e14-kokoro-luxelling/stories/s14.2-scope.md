# S14.2 — /kokoro-luxury-assess

## Objective

Create the `/kokoro-luxury-assess` skill that runs the Triangulo F-S-E
positioning assessment. Classifies an invitado's business as luxury, premium,
or standard based on 5 conversational questions. Persists the result in
`ClientProfile.metadata["positioning_tier"]`.

## Acceptance Criteria

- [ ] Skill file created at `.claude/commands/kokoro-luxury-assess.md`
- [ ] Mirrored to `extension/.claude/commands/kokoro-luxury-assess.md`
- [ ] 5 assessment questions presented conversationally (not as form)
- [ ] Scoring: 4-5/5 = luxury, 2-3/5 = premium, 0-1/5 = standard
- [ ] Client resolution from `.kokoro/clients.json` before assessment
- [ ] Persists `positioning_tier` in client metadata
- [ ] Session log persistence included
- [ ] References `luxelling-assessment.md` as knowledge source
- [ ] Follows Kokoro patterns: Proyector strategy, luxurizante vocabulary
