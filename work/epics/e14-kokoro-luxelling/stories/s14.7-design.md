# Design: S14.7 — Luxury Layers Knowledge File

## Design Decision: Additive Context Pattern (AD-5)

Base skills are NOT modified. A single knowledge file provides supplementary
guidance when `positioning_tier = luxury` is detected in the client profile.

### Per-Skill Mapping

| Base Skill | Primary Source Modules | Key Shift |
|---|---|---|
| /kokoro-diagnose | luxelling-assessment | F-S-E positioning check before diagnosis |
| /kokoro-canvas | luxelling-scarcity, luxelling-growth | Scarcity and distribution constraints |
| /kokoro-finance | luxelling-pricing | Value-based pricing, Veblen Effect |
| /kokoro-pescar | luxelling-communication | Seduce, don't sell; luxury visual codes |
| /kokoro-launch | luxelling-scarcity, luxelling-communication | Slow-burn, invitation-only |
| /kokoro-factory | luxelling-scarcity, luxelling-growth | Hyper-selective distribution |
| /kokoro-funnel | luxelling-experience, luxelling-communication | Seduction funnel, nurturing as ritual |
| /kokoro-mafia | luxelling-quality-symbolism, luxelling-scarcity | Real exclusivity, access rituals |

### Structure per skill section

Blockquote summary, 3-5 numbered considerations with [source module], Key Shift.
Closing Quick Reference Matrix for scanning.
