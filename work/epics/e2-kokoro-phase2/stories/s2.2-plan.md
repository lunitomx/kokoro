### T2: GREEN — Create kokoro-canvas.md skill file

**Goal:** Create the skill file with Eduardo's voice, all 9 blocks in segment-first
order, guiding questions, anti-patterns, and summary template. All T1 tests pass.

**Steps:**

1. Create `extension/.claude/commands/kokoro-canvas.md` with:
   - Opening quote in Eduardo's voice
   - `## Proposito` — why this skill exists
   - `## Antes de Empezar` — Proyector strategy (ask permission, set context, 70/30 rule)
   - `## Bloque 1: Segmento de Invitados` — who you serve, early adopters, guiding questions
   - `## Bloque 2: Problema` — 1-3 real problems, guiding questions
   - `## Bloque 3: Propuesta Unica de Valor` — measurable promise, PUV formula, guiding questions
   - `## Bloque 4: Ventaja Injusta` — what can't be copied/bought, guiding questions
   - `## Bloque 5: Canales` — means of delivering value, guiding questions
   - `## Bloque 6: Flujo de Ingresos` — what you sell + investment + delivery, guiding questions
   - `## Bloque 7: Estructura de Costos` — expense concepts (not amounts), guiding questions
   - `## Bloque 8: Metricas Clave` — 2-3 health numbers, guiding questions
   - `## Bloque 9: Solucion` — LAST, explicit anti-pattern warning, guiding questions
   - `## Resumen del Canvas` — markdown table template for all 9 blocks
   - Reference to `kokoro-phase2-canvas.md` knowledge file
   - Anti-pattern warnings (solution-first bias, "inciensos de ballenas" trap)
   - Eduardo's vocabulary throughout (semilla, raiz, invitados, inversion, creacion)
2. Each block includes:
   - Explanation of what the block captures
   - At least 2 guiding questions (powerful questions, not prescriptions)
   - Eduardo's perspective / coaching notes
   - Interactive prompt: wait for entrepreneur's answer before proceeding

**Verify:**
- `uv run pytest tests/test_canvas.py` → PASS (all GREEN)
- `uv run pytest` → PASS (full suite, no regressions)
- `uv run pyright` → PASS
- `uv run ruff check` → PASS

**Commit:** `feat(s2.2): T2 — GREEN kokoro-canvas.md skill file with Eduardo's voice`
