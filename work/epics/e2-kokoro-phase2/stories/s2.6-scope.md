# S2.6 Scope — Meta Skills Update

> **Story:** S2.6 | **Epic:** E2 | **Size:** S

## User Story

As an entrepreneur who completed Phase 1,
I want the /kokoro router and /kokoro-session to guide me into Phase 2
So that I can continue my journey with the new validation skills.

## In Scope

- Update `kokoro.md` router: add Phase 2 awareness and routing
- Update `kokoro-session.md`: add Phase 2 progress tracking
- Add tests for Phase 2 skill references in both files
- Existing Phase 1 tests must not break

## Out of Scope

- Phase 3/4 routing
- New skill files (already done in S2.2-S2.5)
- CLI changes

## Done When

- Router mentions all 4 Phase 2 skills by command name
- Router can detect Phase 2 readiness and route there
- Session tracks Phase 2 progress (modes: start, continue, review)
- All existing tests pass (no regressions)
- New Phase 2 awareness tests pass
- pyright + ruff clean
