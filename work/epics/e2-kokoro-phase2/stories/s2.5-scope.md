# S2.5 Scope — /kokoro-validate

> **Story:** S2.5 | **Epic:** E2 | **Size:** S

## User Story

As an entrepreneur with validated interviews and mapped forces,
I want a guided validation plan
So that I can design experiments to test my riskiest assumptions systematically.

## In Scope

- Skill file `extension/.claude/commands/kokoro-validate.md`
- 10 Principles of Entrepreneurial Mindset
- 3x3x3 Planning Model (3 years, 3 months, 3 weeks)
- Risk identification from Lean Canvas (Product, Consumer, Market)
- Experiment design with hypothesis template
- Traction concept
- Validation Plan Canvas (Leanstack)
- Eduardo's voice, guiding questions, summary template
- Reference to `kokoro-phase2-validation.md` knowledge file
- TDD with structural content validation
- CLI integration

## Out of Scope

- Actual experiment execution (Phase 3)
- Landing page creation
- Campaign management
- Metric dashboards

## Done When

- Skill file exists at correct path
- 3x3x3 model, risk categories, experiment design present
- Eduardo's voice >= 3 terms
- Summary table >= 4 rows
- All tests pass, pyright + ruff clean
