# Scope: S32.1 — Lean Landing Page Knowledge File

## In Scope

- Extract and structure the complete Lean Landing Page methodology from PDF v2.0
  and class transcript (session 56)
- Document the 5 fundamental principles with verifiable criteria per principle
- Document the 9 skeleton blocks with golden rules per block (objective, rule,
  evaluation criteria, anti-pattern, example)
- Document anti-patterns (what a Lean Landing Page is NOT)
- Include reference metrics (conversion, ICP match, show-up, time to decision)
- Install the knowledge file in both mirrors:
  - `extension/.claude/knowledge/kokoro-lean-landing.md`
  - `.claude/knowledge/kokoro-lean-landing.md`
- Ensure format consistency with existing knowledge files

## Out of Scope

- Skill command creation (`/kokoro-landing`) — that is S32.2
- Test creation — that is S32.3
- Visual evaluation criteria (colors, typography, layout)
- Landing page generation (that is `/kokoro-launch`)
- Modification of existing skills
