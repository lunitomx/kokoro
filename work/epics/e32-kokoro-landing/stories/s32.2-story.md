# S32.2 — Skill: /kokoro-landing

## User Story

As Eduardo (Kokoro operator), I want a `/kokoro-landing` skill that audits
landing pages against the Lean Landing Page methodology, so that entrepreneurs
get actionable strategic feedback on their landing page's decision sequence.

## Acceptance Criteria

1. Receives URL (via WebFetch) or pasted HTML as input
2. Maps landing page content to the 9 structural blocks of the Lean Landing Page
3. Scores each of the 5 fundamental principles (present / absent / partial + diagnosis)
4. Provides rewrite suggestions in Eduardo's voice (luxurizante vocabulary)
5. Suggests `/kokoro-audit` for technical complement when relevant
6. Produces structured output: block map + scorecard + rewrites + top 3 priority actions

## Dependencies

- S32.1 (knowledge file) — COMPLETE
