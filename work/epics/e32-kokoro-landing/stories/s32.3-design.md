# S32.3 — Story Design: Tests + verificacion

## Problem

The `/kokoro-landing` command file was created in S32.2 but has not been
verified against quality gates. Before closing E32, we need confidence that
the skill is production-ready: vocabulary-compliant, structurally complete,
knowledge-accurate, properly integrated, and consistently mirrored.

## Value

A documented verification report provides confidence that `/kokoro-landing`
meets the same quality standard as existing Kokoro skills. Catching issues
now avoids broken experiences when the skill is used with real entrepreneurs.

## Approach

Run 5 verification checks against the command file and document results as
a verification report. No code to write — this is a pure audit story.

**5 Verification Checks:**

| # | Check | Method |
|---|-------|--------|
| 1 | Vocabulary compliance | Grep for forbidden words in instructional content (excluding vocabulary reference table and anti-patrones "don't say" examples) |
| 2 | Output structure | Verify template contains: Mapa de Bloques table, Scorecard de Principios table, Diagnostico por Bloque section, Top 3 Acciones section |
| 3 | Knowledge accuracy | Verify all 9 blocks match `kokoro-lean-landing.md` (names + order), all 5 principles match |
| 4 | Integration | Verify `/kokoro-audit` is suggested as complement, not dependency |
| 5 | Mirror consistency | Diff `extension/.claude/commands/kokoro-landing.md` vs `.claude/commands/kokoro-landing.md` |

## Acceptance Criteria

| ID | Criterion |
|----|-----------|
| AC1 | All 5 verification checks executed and documented |
| AC2 | Each check has explicit PASS/FAIL status |
| AC3 | Verification report committed as story artifact |

## Components

| Component | Action | Path |

## Decisions

| ID | Decision | Rationale |
|----|----------|-----------|
| D1 | No automated test scripts | Out of scope per s32.3-scope.md — manual verification is sufficient for a single command file |
| D2 | Exclude vocabulary table and anti-patrones from forbidden word check | These sections intentionally list forbidden words as "don't say" examples |
