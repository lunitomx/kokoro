# Brief: E31 — Kokoro Creative Review

## Hypothesis

Los emprendedores crean creativos (imagenes, logos, videos, piezas graficas) sin
entender como los evalua el ecosistema de IA de Meta (GEM, Andromeda, Lattice,
Sequence Learning). Un skill que analice creativos bajo estos criterios — con la
voz de Eduardo — les dara retroalimentacion accionable antes de invertir en pauta.

## Success Metrics

- El skill recibe una imagen del usuario y genera un analisis estructurado
- El analisis cubre los 4 sistemas de Meta AI (GEM, Andromeda, Lattice, Sequence)
- Incluye prediccion de rendimiento por tipo de audiencia
- Sugiere mejoras concretas con vocabulario luxurizante
- Knowledge files documentan el ecosistema Meta AI como referencia

## Appetite

Medium — 1 skill command + 1 knowledge file consolidado + integracion con skills
existentes (kokoro-ads, kokoro-creative). 1-2 sesiones.

## Rabbit Holes

- No construir un modelo predictivo real — la prediccion es heuristica informada
- No integrar con Meta Ads API para validar — eso es kokoro-analytics
- No redisenar kokoro-creative — este skill ANALIZA, no genera
