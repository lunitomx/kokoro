# Plan: S8.1 — Skill kokoro-ads

## Tasks

### Task 1: Knowledge file kokoro-ads-meta.md

Create `.claude/knowledge/kokoro-ads-meta.md` documenting Meta Ads platform
constraints and Kokoro vocabulary rules for ad copy generation. Contents:

- Meta Ads character limits (headline 40, primary text 125 visible / 2000 max, description 30)
- Headline best practices (short, benefit-oriented, Kokoro vocabulary)
- Primary text best practices (paragraph format, line breaks, hook in first line)
- WhatsApp Business template structure (greeting message + FAQ buttons)
- Advantage+ audience description format (descriptive, not demographic)
- Kokoro vocabulary table (inversion not precio, creacion not producto, invitado not cliente, etc.)
- Anti-patterns for Meta Ads copy (no markdown, no emojis, no clickbait, no urgency tactics)
- Output format rules (.txt plain text, line breaks for readability)

Mirror to `extension/.claude/knowledge/kokoro-ads-meta.md` (identical copy).

- Files: `.claude/knowledge/kokoro-ads-meta.md`, `extension/.claude/knowledge/kokoro-ads-meta.md`
- Commit after task

### Task 2: Command file kokoro-ads.md

Create `.claude/commands/kokoro-ads.md` following the Kokoro skill pattern
(modeled after kokoro-launch.md). Structure:

- Header with skill name, phase description ("Herramienta: Meta Ads — Creativos y Copy"), Eduardo quote
- Contexto section: purpose, creative-first philosophy, reference to `kokoro-ads-meta.md`
- Contexto previo: read `.kokoro/state.json` if exists, read client `contexto.md`
- Estrategia del Proyector: ask permission before guiding
- Paso 1 — Describir el Creativo: detailed description of visual elements (colors, layout, text, imagery, mood, audience signals)
- Paso 2 — Leer Contexto del Invitado: read contexto.md from campaign folder; fallback (Paso 2b) when no contexto.md exists — ask for client name, project, audience
- Paso 3 — Identificar Audiencia: determine target audience from creative + context
- Paso 4 — Generar Entregables: generate 5 headlines, 5 primary texts, WhatsApp template, Advantage+ audience description
- Plantilla de Salida: .txt output template block (NOT markdown tables) matching the format from the design document
- Nomenclatura: auto-increment file naming (creativo-01, creativo-02)
- Notas para Claude: vocabulary enforcement, anti-patterns, format rules, no emojis, respond in user language
- Persistencia: save to client campaign folder, update `.kokoro/state.json`

Mirror to `extension/.claude/commands/kokoro-ads.md` (identical copy).

- Files: `.claude/commands/kokoro-ads.md`, `extension/.claude/commands/kokoro-ads.md`
- Depends on: Task 1 (command references knowledge file)
- Commit after task
